Training Operators

Train a model using Kubeflow on AWS and AWS Deep Learning Containers

AWS Deep Learning Containers are framework-optimized deep learning environments for training and serving models. Use AWS Deep Learning Containers to optimize your training peformance and training workloads with Training Operators and Kubeflow on AWS.

For CPU, GPU, and distributed GPU tutorials, see Kubeflow on AWS Training in the AWS Deep Learning Containers Developer Guide.

For more information on available images, see Release Notes for Deep Learning Containers.
